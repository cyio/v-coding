# v-coding


### 主要特性

* 即时过滤项目
* 项目有代码，提交历史，任务，合并请求等快捷访问链接
* 简单的任务处理，创建、编辑标题，切换完成状态，删除等（仅私有项目）
* 未读消息Chrome扩展图标提示（由 [fnsoxt](https://github.com/fnsoxt) 提供）
